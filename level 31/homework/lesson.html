ინტერნეტი ინტერნეტი არის გლობალური ქსელი, რომელიც დაკავშირებულია მილიონობით კომპიუტერებთან და მოწყობილობებთან. ის აძლევს საშუალებას მომხმარებლებს გამზარდონ ინფორმაცია,ურთიერთქმედონ და გამოიყენონ მრავალფეროვანი სერვისები. ინტერნეტის ძირითადი ფუნქციები: კომუნიკაცია: ელფოსტა, ჩათი, სოციალური ქსელები. ინფორმაციის გაზიარება: ვებ-გვერდები, ბლოგები, ფორუმები. სერვისები: ონლაინ ბანკინგი, შოპინგი, განათლება. Web სერვერი Web სერვერი არის პროგრამული ან აპარატურული სისტემა, რომელიც პასუხობს ვებ-გვერდების მოთხოვნებს. მისი ძირითადი ფუნქციაა ვებ-გვერდების მომხმარებლებისთვის მიწოდება. Web სერვერის ფუნქციები: HTTP მოთხოვნების მიღება და პასუხის გაცემა. ვებ-გვერდების ჰოსტინგი: HTML, CSS, JavaScript ფაილების მიწოდება. მონაცემთა ბაზებთან კომუნიკაცია: მომხმარებლის მონაცემების მართვა. როგორ მუშაობს? მოთხოვნის გაგზავნა: მომხმარებელი ბრაუზერში შეაქვს ვებ-გვერდის მისამართი (URL). ბრაუზერი აგზავნის HTTP მოთხოვნას web სერვერისკენ. მოთხოვნის დამუშავება: web სერვერი იღებს მოთხოვნას. ის ამოწმებს მოთხოვნილი ფაილების არსებობას და საჭიროებისამებრ პასუხობს. პასუხის გაგზავნა: თუ ფაილები მოიძებნა, web სერვერი აბრუნებს HTML, CSS, JavaScript ფაილებს ბრაუზერში. ბრაუზერი აფორმატებს და აჩვენებს ვებ-გვერდს მომხმარებელს
